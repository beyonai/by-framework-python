# Security Policy

## Supported Versions

Currently, only the following versions are receiving security updates:

| Version | Supported |
| --- | --- |
| 0.2.x | :white_check_mark: |
| < 0.2.0 | :x: |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability, please do NOT create a public Issue.

Please report vulnerabilities by following these steps:

1. Send an email to: `haojingbyai@163.com`.
2. Please include the following details in your email:
   - Type of vulnerability (e.g., Buffer Overflow, SQL Injection, etc.)
   - Steps to reproduce
   - Potential impact
3. We will acknowledge receipt of your report within 48 hours.
4. We will coordinate the disclosure details and timing with you before releasing a public patch, and we will credit your contribution.

Thank you for helping us keep the community safe!
